Folder containing the source code for the dense-dens multiplications.

To launch the experimentations, please type:

    $ cd Constant_Time_AVX_HQC/
    $ ./all.sh
    $ cd Constant_Time_AVX_HQC_gf2x/
    $ ./all.sh
    $ cd Constant_Time_AVX_HQC_TC/
    $ cd hqc-192-2/
    $ make hqc-192-2
    $ bin/./hqc-192-2
    cd ../hqc-256-3/
    $ make hqc-256-3
    $ bin/./hqc-256-3
